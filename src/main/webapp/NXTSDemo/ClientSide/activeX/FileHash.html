== 파일 Hash 값 생성==
Hash란 주어진 정보를 특정 길이의 bit로 만들어 사람들이 Hash함수의 결과로 원래의 정보를 알아 낼 수 없게 하고, 또한 같은 Hash value 값을 같은 두 개의 다른 정보를 찾기 어렵도록 설계된 암호학적으로 안전한 Hash함수 입니다.

Hashing 알고리즘은 입력된 정보의 길이에 따라 두 가지 Hashing 방법을 지원합니다.

입력된 정보의 길이가 2048bit 이하 일 때는 hash()함수를 이용하여 한 번에 해시 값을 계산할 수 있는 방법과 정보의 길이가 2048bit보다 큰 길이를 가질 때는 정보의 길이를 나누어 initialize_Hash()함수로 초기화 하고 나누워진 정보를 process_Hash()함수로 여러 번 수행하여 마지막에 finalize_Hash()함수에 의해 해시 값을 계산하는 방법을 제공하고 있습니다.

지원하는 Hash 알고리즘은 MD5, HAS160, SHA1, SHA256, SHA384, SHA512를 지원합니다.
